# CampusMate - Study Task Manager

## Overview

CampusMate is an Android mobile application designed to help students manage their academic activities efficiently. The application provides task management features together with useful tools such as weather information and book search to improve student productivity and organization.

The application is developed using Java and Android Studio and follows a simple and user-friendly mobile interface.

## Features

### Task Management

* Create study tasks
* Edit existing tasks
* Delete tasks
* Track pending and completed tasks
* Organize daily academic activities

### Weather Information

* View current weather conditions
* Search weather by location
* Access weather information through an external API

### Book Search

* Search books using an external API
* View book details
* Save favorite books for future reference

### User-Friendly Interface

* Simple navigation
* Mobile-friendly design
* Easy access to application features

## How to Run

### Clone Repository

```bash
git clone https://github.com/harindimaheshika710-cell/CampusMate-Study-Task-Manager.git
```

### Open Project

1. Open Android Studio
2. Select Open Project
3. Choose the project folder

### Build Project

Allow Gradle to download dependencies and sync the project.

### Run Application

1. Connect an Android device or launch an emulator
2. Click the Run button in Android Studio
3. Install and launch the application
